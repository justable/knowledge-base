## 如何模拟接口数据

使用 umi 时，可以 mock api 数据，前端可以脱离后端的开发进度，自行测试数据展示效果。那么如果是个 webpack 项目应该怎么配置 mock api 的特性呢？很简单如下所示：

```ts
// webpack.config.js
const apiMocker = require('mocker-api');
module.exports = {
  devServer: {
    before(app) {
      apiMocker(app, path.resolve('./mock/index.js'));
    },
  },
};
```
